### Getting started with GenAI as a student

When using GenAI, always keep the basic principles and general guidelines in mind! For instance, copyrighter material, sensitive or personal data, Non-Disclosure Agreement, embargo master's thesis, ...

Guidelines for students

#### On this page

- What is allowed and how can you be transparent?

- What is not allowed?

### What is allowed and how can you be transparent?

The use of GenAI below is allowed unless it is explicitly stated that it is not allowed within an assignment, course or study programme (for achieving certain learning outcomes).

- Using GenAI as a language assistant for reviewing or improving texts you wrote yourself , provided that the model does not add new content. In this case, the use of GenAI is similar to the spelling and grammar check tools we already have today, so you do not need to explicitly mention using GenAI for this. If you include a claim form (such as the GenAI code of conduct ), of course you mention it there. Again, you always make sure that the use of GenAI does not interfere with an examiner's evaluation of your achieved learning goals. However, from the perspective of academic integrity, you are expected to refer to the original source or author when reviewing or improving texts with GenAI that you did not write yourself (without such a reference, you are committing plagiarism ).

- Using GenAI as a language assistant for paraphrasing text . You can let GenAI paraphrase text you have written yourself or from another author to draw inspiration from it. In that case, there is no need to mention the use of GenAI. If you include a claim form (such as the GenAI code of conduct ), of course you mention it there. However, from the perspective of academic integrity, you are expected to refer to the original source or author when paraphrasing texts you did not write yourself (without such a reference, you are committing plagiarism ).

- Using GenAI as a translation tool for self-written texts. This use is similar to the use of other translation software, so you do not need to explicitly mention using GenAI for this. If you include a claim form (such as the GenAI code of conduct ), of course you mention it there. However, if it involves translating text that you did not write yourself, you should refer to the original source or author (without such a reference, you are committing plagiarism ). Always check the translated text for correctness and meaning.

- Using GenAI as a search engine to get initial information on a topic or to make an initial search for existing research on the topic. This way of gathering information is similar to using an ordinary search engine when working on an assignment. Be aware that the output of a GenAI tool is not 100% reliable, for instance because the tool's underlying database is not complete and/or up-to-date. GenAI can also generate false references. Based on the results obtained, always look for the scientific sources and conduct your own analysis of the source documents. Interpret, analyse and process the information you obtained; don't just copy-paste it. If you then write your own text, you do not have to mention you used GenAI. If you include a claim form (such as the GenAI code of conduct ), of course you mention it there. However, you are expected to reference the scientific sources used; so look for the original sources (after all, without such a reference, you are committing plagiarism ).

- Use where you copy text verbatim. Cite the copied text (with quotes) and make references via a reference list (for this, follow the instructions around referencing GenAI ), footnote or as a personal communication. Explain the use in the methods section (which describes your methodology; if there is such a section). If necessary, include prompts with the complete output, or history, as an attachment or via a link.

- Using GenAI to generate code. This is allowed provided the use of GenAI is mentioned for the copied pieces of code. Explain the use in the methods section (which describes your methodology; if there is such a section). If necessary, include prompts with the complete output, or history, as an attachment or via a link.

- Using GenAI to generate images, audio or video . This is allowed provided the use of GenAI is mentioned . Explain the use in the methods section (if there is one). If necessary, include prompts with the complete output, or history, as an attachment or via a link. If you use existing images, audio or video, you must refer to the original sources used (without such a reference, you are committing plagiarism ).

- Using GenAI to generate new (research) ideas . This is allowed provided the use of GenAI is mentioned . Check whether the idea is actually new or not. There is a good chance it is based on existing work. If that is the case, you should refer to the existing work (if not, you are committing plagiarism ). Always check the accuracy and coherence of the output and engage with it critically.

- Using GenAI to generate synthetic data. This is allowed provided the use of GenAI is mentioned . It is important to carefully evaluate the generated synthetic data for quality and possible bias as the output is highly dependent on the quality of the data on which the models are trained.

- When it is strictly necessary that you copy certain parts of GenAI output verbatim (blocks of text) due to the nature of the assignment (for instance, blocks of text generated by GenAI, other than the above-mentioned authorised use in the context of correcting, translating or paraphrasing self-written text; or elements of a conversation with a GenAI tool), you correctly reference the use of GenAI. Where necessary, cite your original sources . Limit such GenAI use to an absolute minimum.

- If you wish to use GenAI in any other way , always check in advance with the teaching team or your supervisor whether such use is permitted and the conditions, if any, that apply. Clearly explain the purpose, added value and relevance of the desired use. Also state how the use is in line with the assignment, article 84 of the Education and Examination Regulations and the KU Leuven guidelines around responsible use of GenAI .

Note! The output generated by GenAI tools is not always correct and may differ on each attempt. Checking through other sources is always recommended. Related concepts: Tips and considerations for responsible use

### What is not allowed?

- Any form of copy-pasting of any content generated by GenAI without fully acknowledging the source (citing, referencing). After all, if you want to produce any academic work, you need to check the original sources. This does not include the authorised use of GenAI for self-written text as listed under “ What is allowed? ”.  As also indicated above, verbatim copying with reference to GenAI use can only be done in exceptional situations where it is clearly understood in advance that this is in line with the assignment.

- Any use of GenAI that prevents the teaching staff member from checking and/or assessing your skills. For example: using paraphrasing tools on texts you did not write yourself to cover up plagiarism, misuse of translation software.

- Using synthetic data unethically to manipulate research.

- Any use of GenAI during on-campus examinations or other forms of assessment when the teaching staff have indicated that the use of GenAI is not allowed .

Note! When your master's thesis is under embargo , you should be very careful about using AI . After all, any information you enter - for instance, asking the tool to improve the writing of a piece of text - is stored by these systems and can thus be circulated to other users. This is definitely not allowed . If there is an embargo, and you are considering using GenAI, always contact your supervisor first.
